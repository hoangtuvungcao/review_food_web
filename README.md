# Cẩm nang ẩm thực đường phố

Web tĩnh cho Chủ đề 19 - Nhóm 1.

## Thành viên

- Nguyễn Văn Trọng - Nhóm trưởng
- Lê Thuận
- Trần Văn Trường

## Cách chạy

Mở `index.html` trực tiếp bằng trình duyệt hoặc chạy server tĩnh:

```bash
python3 -m http.server 4173
```

Sau đó mở `http://127.0.0.1:4173`.

## Chức năng

- Bản đồ Leaflet với marker màu đỏ cho từng địa điểm.
- Popup bản đồ có mũi tên và nút đóng `×`.
- Tìm kiếm quán ăn theo tên hoặc mô tả.
- Lọc danh sách theo gợi ý gần đây và top trending.
- Xem chi tiết địa điểm, đánh giá sao và viết review.
- Thêm địa điểm mới, chọn vị trí trên mini-map và lưu tạm bằng `localStorage`.
- Thêm quán mới bắt buộc có ảnh/logo quán.
- Review có thể gửi kèm ảnh món ăn tại quán.
- JavaScript dùng jQuery cơ bản để thao tác giao diện.
