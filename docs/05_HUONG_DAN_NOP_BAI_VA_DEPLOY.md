# Hướng dẫn nộp bài và deploy

## 1. Chuẩn bị mã nguồn GitHub

1. Tạo repository chung trên GitHub.
2. Đưa toàn bộ mã nguồn vào repository.
3. Kiểm tra repository có các file chính:
   - `index.html`
   - `about.html`
   - `css/style.css`
   - `js/main.js`
   - `images/logo/logo.svg`
   - `favicon.ico`
   - `README.md`
   - thư mục `docs/`

## 2. Deploy bằng Netlify

1. Đăng nhập Netlify.
2. Chọn `Add new site`.
3. Chọn `Import an existing project`.
4. Kết nối GitHub repository.
5. Vì là web tĩnh, để trống build command.
6. Publish directory: `.`.
7. Deploy và lấy link website.

## 3. Deploy bằng Vercel

1. Đăng nhập Vercel.
2. Chọn `Add New Project`.
3. Import GitHub repository.
4. Framework preset: `Other`.
5. Build command: để trống.
6. Output directory: `.`.
7. Deploy và lấy link website.

## 4. Chuẩn bị báo cáo

### File `.docx`

Nội dung nên gồm:

- Giới thiệu đề tài.
- Phân tích yêu cầu.
- Thiết kế giao diện đa nền tảng.
- Phân công nhiệm vụ nhóm.
- Mô tả chức năng hệ thống.
- Công nghệ sử dụng.
- Kiểm thử và đánh giá kết quả.
- Hướng phát triển.

### File `.pptx`

Slide nên gồm:

1. Tên đề tài và thành viên nhóm.
2. Lý do chọn đề tài.
3. Yêu cầu hệ thống.
4. Phân công nhiệm vụ.
5. Giao diện chính.
6. Chức năng bản đồ và popup.
7. Chức năng thêm quán và review.
8. Kết quả đạt được.
9. Hạn chế và hướng phát triển.

### Video demo 3-5 phút

Kịch bản demo đề xuất:

1. Giới thiệu website.
2. Mở trang chủ.
3. Tìm kiếm quán ăn.
4. Bấm danh sách để di chuyển tới marker.
5. Mở popup và xem chi tiết.
6. Thêm review kèm ảnh món ăn.
7. Thêm quán mới kèm ảnh/logo.
8. Giới thiệu trang tổng quan và phân công nhóm.

## 5. Nộp bài trên LMS

1. Upload file `.docx`, `.pptx` và video demo lên Google Drive.
2. Bật quyền chia sẻ link.
3. Chuẩn bị các link:
   - Link GitHub repository.
   - Link website deploy.
   - Link Google Drive báo cáo.
4. Nộp các link lên eLearning/LMS theo yêu cầu giảng viên.
